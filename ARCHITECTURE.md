## 三、文件上传 → 研判流程

上传文件 (图片/TXT/DOCX)
     │
     ├── 
┌─────────────┐
│ handleBefore│
│ Upload      │
│ 前端验证+预览│
└──────┬──────┘
       │ 点击"开始识别分析"
       ▼
┌─────────────────────────────────────────────┐
│ startImageAnalysis                          │
│ ├── 图片 → POST /api/ocr → EasyOCR 提取文字  │
│ ├── TEXT  → 直接读取内容                     │
│ └── DOCX  → POST /api/extract-text → 解析    │
└──────────────────┬──────────────────────────┘
                   │ 提取文字填入 inputText
                   ▼
┌─────────────────────────────────────────────┐
│ startAnalysis (自动触发 AI 研判全流程)        │
└─────────────────────────────────────────────┘
